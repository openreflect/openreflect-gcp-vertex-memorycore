# Google Cloud CLI (gcloud) Documentation

This directory contains the Google Cloud CLI (gcloud) documentation optimized for AI retrieval.

## Files

- `gcloud-cli-documentation.json` - Complete gcloud CLI documentation in structured JSON format

## Structure

The JSON file is organized into the following sections:

1. **metadata** - Source information and version details
2. **overview** - Description and purpose of gcloud CLI
3. **installation** - Installation instructions for various platforms
4. **authentication** - Authentication methods and account management
5. **configuration** - Configuration management and common properties
6. **globalFlags** - All available global flags with descriptions
7. **commandStructure** - Command format and available groups
8. **outputFormats** - Available output formats (json, yaml, table, etc.)
9. **keyCommands** - Essential commands organized by category:
   - Authentication
   - Configuration
   - Project Management
   - Services
   - Compute
   - Cloud Run
   - Vertex AI
   - IAM
   - Storage
10. **commonWorkflows** - Step-by-step workflows for common tasks
11. **filteringAndFormatting** - Filter and format options
12. **components** - Component management
13. **topics** - Additional gcloud topic information
14. **bestPractices** - Best practices for using gcloud CLI
15. **scripting** - Scripting examples and patterns
16. **errorHandling** - Common errors and debugging techniques
17. **resources** - Links to official documentation
18. **index** - Keywords and categories for search optimization

## Key Features

### Authentication
- User account authentication
- Application Default Credentials (ADC)
- Service account authentication

### Configuration
- Project, region, and zone settings
- Named configurations for multiple environments
- Property management

### Common Commands
- Project management: `gcloud projects list`, `gcloud config set project`
- Service management: `gcloud services enable`, `gcloud services list`
- Compute resources: `gcloud compute instances list`
- Cloud Run: `gcloud run services list`, `gcloud run deploy`
- Vertex AI: `gcloud ai models list`, `gcloud ai endpoints list`
- IAM: `gcloud iam service-accounts list`, `gcloud projects add-iam-policy-binding`
