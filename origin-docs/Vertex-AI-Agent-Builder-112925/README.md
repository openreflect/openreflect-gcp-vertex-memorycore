# Vertex AI Agent Builder Documentation

This directory contains the Vertex AI Agent Builder documentation compiled from https://docs.cloud.google.com/agent-builder/overview and optimized for AI retrieval.

## Files

- `vertex-ai-agent-builder-documentation.json` - Complete documentation in structured JSON format

## Structure

The JSON file is organized into the following sections:

1. **metadata** - Source information and version details
2. **overview** - Description and key capabilities
3. **components** - Detailed breakdown of Build, Scale, and Govern components:
   - **Build**: Agent Development Kit (ADK), Agent Garden
   - **Scale**: Vertex AI Agent Engine, Agent Tools
   - **Govern**: Security Command Center, Agent Identity
4. **workflow** - Step-by-step workflow for building and deploying agents
5. **documentationStructure** - Complete documentation structure including:
   - Agent Development Kit topics
   - Vertex AI Agent Engine topics
   - Sessions, Memory Bank, Code Execution
   - Example Store, Agent2Agent Protocol
   - Agent Tools categories
6. **resources** - Links to documentation, blog posts, and next steps
7. **keyConcepts** - Important concepts and terminology
8. **index** - Keywords, components, frameworks, and tools for search optimization

## Key Components

### Agent Development Kit (ADK)
Open-source framework for building sophisticated multi-agent systems.

### Agent Garden
Library in Google Cloud console with prebuilt agents and tools (Preview).

### Vertex AI Agent Engine
Set of services for deploying, managing, and scaling AI agents in production.

### Agent Tools
- Built-in tools (Google Search, Vertex AI Search, Code Execution)
- Google Cloud tools (APIs, Integration Connectors, Application Integration)
- MCP tools (Model Context Protocol)
- Ecosystem tools (LangChain, CrewAI, GenAI Toolbox)
