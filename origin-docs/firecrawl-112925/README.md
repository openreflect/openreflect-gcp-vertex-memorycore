# Firecrawl MCP Documentation

This directory contains the Firecrawl MCP Server documentation compiled from https://docs.firecrawl.dev/mcp-server and optimized for AI retrieval.

## Files

- `firecrawl-mcp-documentation.json` - Complete documentation in structured JSON format

## Structure

The JSON file is organized into the following sections:

1. **metadata** - Source information and version details
2. **overview** - Description and key features
3. **installation** - Installation instructions for various platforms (Cursor, Windsurf, VS Code, etc.)
4. **configuration** - Environment variables and configuration examples
5. **tools** - Complete documentation for all 8 available tools:
   - `firecrawl_scrape` - Single URL scraping
   - `firecrawl_batch_scrape` - Batch URL scraping
   - `firecrawl_check_batch_status` - Check batch operation status
   - `firecrawl_map` - Website mapping and URL discovery
   - `firecrawl_search` - Web search with optional content extraction
   - `firecrawl_crawl` - Asynchronous website crawling
   - `firecrawl_check_crawl_status` - Check crawl job status
   - `firecrawl_extract` - LLM-powered structured data extraction
6. **features** - Rate limiting, credit monitoring, batch processing
7. **logging** - Logging system details
8. **errorHandling** - Error handling capabilities
9. **development** - Development and contribution guidelines
10. **resources** - Links to API keys, GitHub, documentation
11. **index** - Keywords and tools for search optimization

## Usage

This documentation is optimized for:
- AI/LLM context retrieval
- Programmatic access
- Search and filtering
- Tool reference lookup
